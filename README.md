# College Major Selector

This is a web app that helps students figure out what college major suits them best.

## What it does
- Users answer questions about their interests and strengths
- The app recommends majors that align with student profiles
- Built with Replit

## Why I built it
I noticed a lot of students, including me, struggle to choose a major, so I wanted to 
build something that makes that process easier and more personalized.
